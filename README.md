# Global-Screening

Dashboard market dengan UI modern untuk memantau:
- Saham Indonesia
- Crypto market cap terbesar
- Macro indicators

## Fitur
- Real-time fetch (auto-refresh tiap 60 detik + tombol manual refresh)
- Pencarian dan sorting data
- Chart mode (`Line` / `Bar`) per asset
- Tema dark/light
- Ringkasan market (asset naik, turun, rata-rata perubahan)

## Data Source
- Yahoo Finance (quote + short history) untuk saham/macro
- CoinGecko API untuk crypto

> Jika provider API mengalami limit/CORS, app akan menampilkan data terakhir yang tersedia.
